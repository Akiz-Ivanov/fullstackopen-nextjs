# BlogList

A blog-sharing app built with **Next.js**, **PostgreSQL**, and **Drizzle ORM** as part of the [Full Stack Open](https://fullstackopen.com/) curriculum.

## What you can do here

- **Share blogs** — post links to articles you've found, with title, author, and URL
- **Like posts** — upvote blogs you enjoyed
- **Build a reading list** — every blog you add gets tracked automatically, and you can bookmark others' posts to read later
- **Generate an API token** — access your data programmatically via `/api/me`

## Getting started

1. <Link href="/register">Register</Link> an account
2. <Link href="/login">Log in</Link>
3. <Link href="/blogs/new">Create a new blog</Link>
4. Browse <Link href="/blogs">all blogs</Link> or check out other <Link href="/users">users</Link>
